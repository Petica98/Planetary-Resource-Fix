The plan is mapping resource deposists as accurately as possible in a maner that it can be used in games etc
Most of the number I'm using come from NASA and ESA, with some exceptions for I think the indian space agency as an additional source

To accurately map which real life resource should correspond to in game resources.
Aside from a potential add of a grouping of Sulphates/chemical processes everything fit pretty neatly into existing Resources

For now regolith will be concidered a low yield for metals and silicon (0.03) and lower for the o2 trapped in it (0.01) 
In future patches I might compatibility patch with tedds regolith mod, but I think thus far it only has metalic regolith and not the others
I do also think regolith makes more sense as a deposit type instead of a resource itself looking at the current game

Atmosphere I will tend to keep mass the same as is in game as to not absolutely destroy any hope of terraforming with a factor per planet
Ground resources I tend to devide by 1,000,000 (1 million) as to bring teratons to metatons, a more balance-able number usually
